藤四郎的夏天

Posted on 2016-07-06 | Edited on 2016-07-17 | In Text

（一）

  本丸向来无四季，所谓模糊的季节观念，也就是来自于审神者携带的礼物。当审神者带来的点心从一颗颗鲜红的草莓变做一筐筐艳丽的樱桃再到切的极整齐的芒果再到芒果雪冰，季节也就有了些更具象的意义。

  鹤丸是第一把发现夏天到来的刀。

  那日他恰好在门口当值，拄着扫帚百无聊赖时，门外传来极轻的脚步声，然后是重物落地的声音。

  他把扫帚放在一边，带上兜帽打开门探出半个身子来：“今天被勤奋的我吓到了吗？”话音未落本丸的主人已经施施然跨进门来，鹤丸的视线自然的落在审神者的脚边，探寻着这次带回的是怎样的伴手礼，一双款式新奇的露趾的鞋不出意料的进入他的视线，接着是裸露的脚踝，笔直的小腿，有些旧疤的膝盖…

在视线忍不住上移到大腿线条之前，鹤丸以一把太刀不应有的机动卸下了自己的白色羽织，把一头雾水的本丸之主从头到脚裹成了一个球，而后长抒一口气：“这可真是吓到我了。”

“鹤丸果然不愧是是平安时代的刀呢。”

措手不及的审神者艰难的从裹成一团羽织里找到了袖口，总算是解放了双手，笑眯眯地呼扇着宽大的袖幅，下一秒加州清光就感应一般的出现了。

察觉到清光在白色羽织上打转的目光，审神者两只手一左一右的提好衣服的下摆，展臂平伸，她今日的装扮也就显露了出来。只看了一眼，这把红色的打刀的脸就像这夏天一样腾的烧起来了。

夏天到来的第一天，审神者是被两把刀护送回房间的，清光拔刀护卫在她身边，鹤丸在前面开路，一手一个的提留着两个大西瓜，听着后方时不时传来“不是说好穿裙子”“太麻烦了不想折腾”的争论声，送一人一刀至房间门口，鹤丸提着西瓜就晃晃悠悠的走向厨房，循着审神者“西瓜冰镇一下再切”的建议找烛台切光忠去了。

这厢审神者已经干脆的卸了碍事的外套，穿着的也不过是一件领口稍大的短袖，一条简单的牛仔热裤，脚上踩一双厚底露趾凉鞋，现世女孩子夏天再普通不过的装扮。

尽管是在网络上看到过很多次的日常穿搭，但是直面活生生的实体个例，冲击力还是毋庸置疑的，是领口露出的肌肤，堪比粟田口家的短裤，尤其是一双长腿，以膝盖上下作为肤色分界，以下是健康的小麦色，以上是少见阳光的白。




不知不觉日子就晃悠悠的来到了七月八日。

七月八日究竟是个何许日子，新来的虎澈清楚，土方家的两位心照不宣早早准备起来，而冲田家的两位，自是不用说。

原属于总司的两位，倒是像什么都没有发生过一样。

安定每日催着审神者出门锻炼，夏日以来更加懒散的审神者负隅顽抗，各位千年辈的爷爷插科打诨，真是“小小姐体力不如大小姐，连胃口也不如大小姐了。”

清光则更是平静，似乎比起正日，审神者打算选购的指甲油才更为重要一些，“哈哈主终于心血来潮准备涂指甲油了啊。”三日月慢条斯理的路过又招呼着新来的“大小姐”远去了“好孩子慢点跑…”

歌仙沉吟半响，觉得终于这词用的妙极，小小姐和加州清光的关系好至此，早晚会画一样的指甲，捣一样的装扮，只不过恰好是近日。

文摘:

  








   






看到果壳问答里有人要求推荐各学科入门书籍，想起以前正好做过类似的工作。
是在2009年的时候，集合了科学松鼠会和豆瓣的网友，一起推荐了这些书。

当时的要求是：“先写下你的专业领域，再回答这个问题‘如果一个受过高中教育、但完全不了解你这个领域的人想学习之，你推荐哪本入门书籍？’”。大量网友参与，最后总结了以下这些书。不过，最后列出来的书籍，恐怕不完全是高中水平能掌控的，也难怪，不是所有领域都能轻易入门的。 
　　 
所以，最后我将这个列表定义为：“对想了解某个领域的初学者来说最值得推荐的一本书”。

学科在演进，新书不断出来。今天整理到这里，也希望果壳网的网友继续推荐，一起来打造一份理想的榜单。
----------------------------------

1. 师从天才
作者 : [美] 罗伯特·卡尼格尔
出版社 : 上海科技教育出版社
评语 : sunfield推荐（药理学）：介绍里说这本书说的是科研界的师承关系，其实大部分讲的是现代药理学的发展史。

2. 普通生物学
作者 : 陈阅增
出版社 : 高等教育出版社
评语 : 桔子（细胞生物学）推荐：我就是高中看《普通生物学》才突然对生物感兴趣。以至于报考时候脑子里除了生物其它什么也不知道。比后来编的所谓什么“面向21世纪教材”强多了；生物学入门的好书；come_god说：记得高三参加全国生物学竞赛时看的第一本书。《普通生物学》高教出版社

3. 什么是数学
作者 : [美] R·柯朗 H·罗宾 著/I·斯图尔特 修订
出版社 : 复旦大学出版社
评语 : 数学专业众人推荐

4. 病者生存
作者 : （美）莫勒姆，（美）普林斯 著，邵毓敏 译
出版社 : 广西科学技术出版社
评语 : BoBo（医学）推荐：从新的角度来看我们所生的疾病。英文原本也不长的。这本中译本，看看既新鲜又产生很多奇怪想法。

5. 物理世界奇遇记
作者 : （美）伽莫夫/（英）斯坦纳德
出版社 : 科学出版社
评语 : bullimit（理论物理）推荐：这本书中对理论物理基础做了极富独创性而又深刻的描述， 不是一般物理类的科普书-例如时间简史所能够比得上的。

6. 追寻记忆的痕迹
作者 : [美] 埃里克·坎德尔
出版社 : 中国轻工业出版社
评语 : 我的推荐（神经科学）：可当作梳理神经科学历史的一本书

7. 梦断代码
作者 : Scott Rosenberg
出版社 : 电子工业出版社
评语 : idel（软件工程）推荐：如果是项目管理和风险控制呢，可以看《梦断代码》；如果只是单纯的追求写程序的艺术呢，可以看《编程之道》；如果是追求技术卓越，可以看《代码大全

8. 编码的奥秘
作者 : Charles Petzold/伍卫国/王宣政/孙燕妮
出版社 : 机械工业出版社
评语 : windstorm（计算机工程，嵌入式）推荐；come_god 说，什么都不说了，大家可以去找来看。神作！

9. 大众天文学(上下两册)
作者 : C.弗拉马里翁
出版社 : 广西师范大学出版社
评语 : Gerry（天文学）的推荐

10. 费恩曼物理学讲义（第1卷）
作者 : Richard P. Feynman/Robert B. Leighton/Matthew Sands
出版社 : 上海科学技术出版社
评语 : Yin Zhangqi推荐（量子光学）：用于学习现代物理学非常有帮助。如果想专门学习量子物理学，可以读第三卷，是入门的好教材；bullimit点评：经典，同时还可以结合阅读费曼的两本自传及QED这本演讲集来作为辅助入门读物。

11. 环境科学：全球关注 上下
作者 : 库宁汉姆
出版社 : 科学出版社
评语 : newgnaw（环境科学）：有中译本，虽然个别地方译的不是太好。比国内的各种教材入的深，出的浅，有趣得多。

12. 最大的小发明：螺丝与螺丝刀
作者 : 里布金斯基
出版社 : 百花文艺出版社
评语 : 霹雳东风（机械专业）推荐：前半部分有些学究考古的味道，没什么意思。后半部分被这个外行人道出了机械行业一个最关键的概念之一——精度。而且从螺丝的角度来讲精度，就相当于是从加工工艺的角度去讲。所以这本书相当靠谱。

像紫薇答应尔康那样…
小贞你爱爱我好吗。



（一）







“眼见他起高楼，



眼见他宴宾客，



眼见他楼塌了…”



婶跟着哼唱，凄凄切切，唱到动情处拉着被禁足的清光，一起装模作样的抹眼泪。



“又怎么了？”安定问在门口抱着手臂的药研。



“大概是在哀悼开启白金台之后失去的资源和手札吧。”药研耸耸肩。



（二）



“不想跟你们混熟，想搞好关系找光忠和小贞去。”



第一天的大俱利也依旧是一个人的大俱利。



婶气的往他盘子里糊了整整一盘花椰菜。



“你有本事说，你有本事带回来啊！”



第二天的大俱利又说了同样的话。



第三天的大俱利也说了同一句话。



…



“其实大咖喱也蛮希望小贞回来的吧。”



一个婶若有所思的说。



（三）



“这新的合战场…充满了各种各样的报社队啊…”



婶深深的叹气，“比如全四花打刀，全大太刀，这种就属于送温暖的队伍。”



“至于全胁差…”



只恨自己不是孙大圣，



“兀那蜘蛛精，吃俺老孙一棒！”



（四）



政府通告，从来是几家欢喜几家忧。



贞酱实装的公告发放下来的当天，本丸就稍稍的沸腾了起来。



光忠拿着婶递过来的钱包去万屋大采购，同事也纷纷道贺。



而婶正在拜托江雪和膝丸去轮班观察一下莺丸的情况。



“怎么办啊好担心莺爷爷…我跟你们讲我今天去给兼桑手入的时候，看到莺爷爷领着一排莺丸徘徊在手入室门口，真的吓死我了…”



“所以拜托了！”



婶拉过膝丸，双手合十。



“啊，交给我吧。”



膝丸自信的说。



“这世间，注定充满了遗憾…”江雪叹了一声，先行去了。







（五）



“光忠光忠，来搭把手呗。”



“好。”



光忠愉快的应着，手底动作不停，一排切的整整齐齐的小青菜很快就叠在了盘里。



“您这是要煮汤面？”



婶微微点头，继续紧张的注视着微沸的水面，谨防扑出锅。



光忠金色的眼睛一转，了然的笑了起来，取出了一个鸡蛋。



“啊，莺丸殿下喜欢的口味，应该是清淡的，煮一个荷包蛋，加一点小青菜的，不要加火腿，放盐这种事情交给帅气的我吧。”



“光忠你真好，我感觉莺爷爷今天要在房间里吃茶水泡白饭了。”



在光忠的指导下，婶突破方便面水平的第一锅卖相极好的汤面成功出锅。



莺丸静坐茶室，闭眼听着脚步声越来越近，停在门口，然后本丸家主的声音响起来，伴着汤的热气涌进寂然的茶室。“莺爷爷，我还没吃饭，你陪我吃个晚饭嘛…”



他睁开眼睛笑着应了。



虽然事后，本来一直不吃晚饭的婶和已经被平野一期江雪髭切膝丸鹤丸投喂了很多点心的莺丸都感觉自己撑了个半死。



（六）



听到门口高跟鞋登登敲击地面的声响，加州清光过五关斩六将的飞奔到了门口，好好一把打刀跑出极化短刀的机动。



“今天是清光给我开门呀？”



站在玄关的婶三两下蹬掉了鞋子，踏上木质地板笑眯眯地张开了双臂。

“不-想-放-开。”



“那我指路，就这么走着去好了。”



清光笑的眯起眼睛，像一只撒娇的猫咪。

婶扶着安定解着系带的高跟鞋抱怨着，然而安定已经完全能预料到后续话题并不想说话了。
